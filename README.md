# ⚡ PokéNext

Um aplicativo web moderno para explorar o universo Pokémon, construído com Next.js 15 e React 19. Pesquise, visualize detalhes e gerencie seus Pokémon favoritos com uma interface rápida e responsiva.

## ✨ Funcionalidades

- 🔍 **Busca em tempo real** - Encontre Pokémon por nome
- 📊 **Detalhes completos** - Visualize estatísticas, tipos e habilidades
- ❤️ **Favoritos** - Salve seus Pokémon preferidos (persistência local)
- ⚡ **Loading otimizado** - Estados de carregamento com Suspense
- 🎨 **Interface moderna** - Design responsivo com Tailwind CSS
- 🌙 **Tema escuro** - Suporte a dark mode
